# Jamaica Driving School Finder 🚗🇯🇲  

> 🚧 Work in progress — early prototype generated with Google AI Studio  
> Helping Jamaican students find the best driving schools through a modern directory and interactive learning tools.

**A modern, gamified directory for driving schools across Jamaica.**  
Built with **Next.js 15**, **React 19**, **TailwindCSS 4**, and optional **Gemini API**.

---

## 🌍 Overview

**Jamaica Driving School Finder** is a modern directory designed to help Jamaican students discover, compare, and choose the best driving schools across the island.

The project includes:

- A **premium directory**
- An **educational test system**
- An **XP and level progression system**
- A **student leaderboard**
- **Role‑based access** (Student / Partner)
- A **modern UI**
- A **professional Next.js architecture**

This project is in **advanced development**, with many features already implemented.

## 🧩 Local Development

### 1. Install dependencies
```bash
npm install
```

### 2. (Optional) Configure Gemini API
Create `.env.local`:
```
GEMINI_API_KEY=your_key_here
```

### 3. Start the development server
```bash
npm run dev
```

App available at:
```
http://localhost:3000
```
